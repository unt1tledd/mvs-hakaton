## О проекте

Веб-приложение, которое позволяет за 30 секунд подключить таблицу MWS Tables к соцсети и получить красивую, понятную аналитику:

- Общая статистика (посты, просмотры, лайки, комментарии)  
- Динамика лайков и просмотров по дням/неделям/месяцам  
- Структура охвата (вирусный, подписчики, реклама)  
- Поддержка нескольких сообществ одновременно  
- Скачивание готовых шаблонов Excel (для ВК и универсальный)

---

## Структура проекта
- api - папка с функциями для взаимодействия с таблицей и фронтом 
- templates_xlsx - шаблоны таблиц для скачивания, чтобы пользователь мог их экспортировать в mws
- frontend - внешняя часть функционала
- parse_vk - функция с парсингом апи vk

к сожалению не до конца успели доделать проект вроде более менее рабочий фуннкционал готов, но не успели связать это с фронт-частью 

Был сценарий взаимодействия:
1. пользователь скачивает шаблоны, подает нам токены и ссылки на таблицы
2. мы парсим апи и далее заполняем таблицу + визуализация стаистики, маштабируемые графики и круговые диаграммы для анализа охватов и тп
